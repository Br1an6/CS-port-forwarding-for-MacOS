# CS-port-forwarding-for-MacOS
Port-forwarding script for CS1.6, CZ on MacOS
- Using homebrew with [MiniUPnP](http://miniupnp.tuxfamily.org)

Simple Run:

`./cs.sh`

If there is an permisson error run (`sudo` if needed):

`chmod +x cs.sh`
