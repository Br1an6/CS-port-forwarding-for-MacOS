# CS-port-forwarding-for-MacOS
Port-forwarding script for Counter-Strike:CS1.6, CZ on MacOS
> Using Homebrew with [MiniUPnP](http://miniupnp.tuxfamily.org)

Simple Run:

`./cs.sh`

If there is any permisson error run ( `sudo` if needed ):

`chmod +x cs.sh`
